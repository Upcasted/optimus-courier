Optimus Courier integrates your WooCommerce store with Optimus Courier service for automated shipping and tracking solutions.

== Description ==

Optimus Courier is a WordPress plugin that seamlessly connects your WooCommerce store with Optimus Courier service. It simplifies your shipping operations and provides tracking capabilities for your customers.

Key Features:

* Automated shipping label generation for Optimus Courier
* Real-time tracking integration
* Order status synchronization
* Easy to use admin interface

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/optimus-courier` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings in the WordPress admin panel under 'Optimus Courier'
4. Enter your API credentials for Optimus Courier service

== Frequently Asked Questions ==

= Which courier services are supported? =

The plugin currently supports Optimus Courier service exclusively.

= How do I get API credentials? =

You can obtain API credentials by contacting Optimus Courier support team.
